## 经验教训

### 遇到的问题
1. CLIP 维度不匹配 (768 vs 3072) - 解决：使用正确的 ClipLoaderGGUF 配置
2. KSampler 参数类型错误 - 解决：使用字符串而非索引值
3. Note 节点导致验证失败 - 解决：过滤非执行节点
4. widgets_values 顺序映射 - 解决：按 inputs 顺序对应无链接的字段

### 最佳实践
1. 优先使用已有的工作流 JSON 文件
2. 保持节点 ID 和连接关系不变
3. 只修改提示词、尺寸、种子等参数
4. 验证 KSampler 等关键节点的参数类型
