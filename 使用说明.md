# 🦞 小龙虾 ComfyUI 控制器 - 使用说明

## 📖 快速开始

**一句话说明：**
> 直接把文件扔给小龙虾，告诉它你要什么，它自动帮你批量生成图片！

---

## 🎯 支持的指令格式

### 方式一：直接描述需求

```
帮我自动完成 10 个图片，图片结合最新新闻的观点，图片尺寸 1024*512
```

小龙虾会自动：
1. ✅ 获取最新热点新闻
2. ✅ 为每个新闻生成专业提示词
3. ✅ 批量执行 10 张图片生成
4. ✅ 自动下载并分类保存

---

### 方式二：上传小说/文本文件

```
[上传小说文件：《斗破苍穹.txt》]
帮我把这本小说的每个章节生成一张插图，尺寸 1024*1024
```

小龙虾会自动：
1. ✅ 读取小说内容
2. ✅ 提取每个章节的关键场景
3. ✅ 为每个场景生成提示词
4. ✅ 批量生成章节插图
5. ✅ 按章节命名保存

---

### 方式三：上传提示词列表

```
[上传文件：prompts.txt]
用这个工作流帮我生成这些提示词的图片
```

**prompts.txt 格式：**
```
一个美丽的女孩，高清，精致，专业摄影
赛博朋克城市夜景，霓虹灯，未来科技感
中国古风山水，水墨画风格，意境深远
...
```

## 🚀 完整指令示例

### 示例 1：热点新闻批量生成

```
帮我自动完成 10 个图片，图片结合最新新闻的观点，图片尺寸 1024*512
```

1. 获取今日热点新闻（10 条）

## ⚙️ 高级参数

### 在指令中指定参数

```
帮我生成 10 张图片，基于最新新闻
- 尺寸：1024*512
- 步数：20
- 风格：写实摄影
- 工作流：Z-Image Turbo
- 输出目录：/path/to/output
```

### 可用参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 尺寸/分辨率 | 图片大小 | 1024*1024 |
| 步数/steps | 采样步数 | 6 (Z-Image) |
| 风格 | 写实/动漫/玄幻等 | 自动识别 |
| 工作流 | 使用的工作流 | Z-Image Turbo |
| 批量数量 | 生成数量 | 自动识别 |
| 输出目录 | 保存位置 | ~/Downloads/comfyui_organized/ |

## 🛠️ 技术说明

### 小龙虾的自动处理流程

```
用户指令
   ↓
[1] 解析意图 → 识别批量生成需求
   ↓
[2] 读取文件 → 提取内容/提示词/章节
   ↓
[3] 生成提示词 → AI 优化每个场景描述
   ↓
[4] 创建工作流 → 配置 ComfyUI API
   ↓
[5] 批量提交 → 逐个执行生成任务
   ↓
[6] 监控进度 → WebSocket 实时反馈
   ↓
[7] 下载整理 → 分类保存 + 元数据
   ↓
[8] 汇报结果 → 汇总成功/失败统计
```

### 文件读取逻辑

**小说/文本文件：**
- 按章节分割（识别"第 X 章"pattern）
- 提取关键场景（战斗/突破/对话/环境）
- 生成画面描述提示词

**提示词列表：**
- 每行一个提示词
- 自动优化质量词（8K/高清/精致等）

**新闻内容：**
- 抓取标题 + 正文
- 提取关键视觉元素
- 生成画面描述

## 📝 最佳实践

### ✅ 推荐做法

1. **明确数量**：「帮我生成 10 张图片」
2. **指定尺寸**：「尺寸 1024*512」
3. **说明风格**：「玄幻修仙风格」
4. **分批执行**：大量任务分批次（每次 10-20 个）

### ❌ 避免做法

1. 一次生成上百张（容易超时）
2. 文件过大（建议分章节）
3. 提示词过于简单（小龙虾会帮你优化）

## 🔧 故障排除

### Q: 生成失败怎么办？
A: 小龙虾会告诉你具体错误，常见原因：
- ComfyUI 未运行 → 先启动 ComfyUI
- 模型缺失 → 自动下载或手动放置
- 显存不足 → 降低尺寸或分批生成

### Q: 如何查看已生成的图片？
A: 小龙虾会汇报保存路径，通常在：
```
~/Downloads/comfyui_organized/
├── news/           # 新闻配图
├── novel/          # 小说插图
├── custom/         # 自定义
└── ...
```

### Q: 如何中断生成？
A: 直接说「停止生成」，小龙虾会立即终止任务。
