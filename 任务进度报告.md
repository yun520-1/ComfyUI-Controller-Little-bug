## 💡 经验总结

### 图片生成成功关键
1. 使用官方工作流 JSON 作为模板
2. 保持节点 ID 和连接关系不变
3. 只修改提示词、尺寸、种子等参数
4. 正确映射 widgets_values 到 inputs

### 视频生成挑战
1. LTX2 工作流更复杂 (40 节点 vs 11 节点)
2. Reroute 节点需要特殊处理
3. 视频生成时间长 (5-15 分钟)
4. 需要更多显存和计算资源

### 最佳实践
1. 优先使用 API/CLI，避免浏览器自动化
2. 先找现有 skill，再考虑自己实现
3. 复杂任务先写计划文件
4. 每步完成后更新进度文档
