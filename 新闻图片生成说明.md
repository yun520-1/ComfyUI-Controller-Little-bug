## 🎯 手动生成步骤

### 步骤 1: 访问 ComfyUI

打开浏览器访问:
```
http://127.0.0.1:8188
```

### 步骤 2: 加载默认工作流

如果界面为空，点击右侧菜单的 **"Load Default"** 加载默认工作流

### 步骤 3: 设置图片尺寸

找到 **EmptyLatentImage** 节点，设置:
- **width**: 1024
- **height**: 512
- **batch_size**: 1

### 步骤 4: 输入提示词

找到 **CLIP Text Encode (Positive)** 节点，输入提示词:

**第一张图片 - 新闻演播室**:
```
professional news broadcast studio, modern TV anchor desk, breaking news banner, 4K ultra realistic, broadcast quality lighting, cinematic, highly detailed
```

**第二张图片 - 新闻标题背景**:
```
digital news headline background, futuristic screen display, latest news ticker, blue and red theme, professional broadcast studio, 4K, highly detailed
```

### 步骤 5: 设置负面提示词

找到 **CLIP Text Encode (Negative)** 节点，输入:
```
blurry, low quality, distorted, ugly, deformed, watermark, text, signature, worst quality, lowres
```

### 步骤 6: 点击生成

点击右侧的 **"Queue Prompt"** 按钮

### 步骤 7: 等待生成

- 生成时间：约 10-30 秒/张（取决于 GPU）
- 生成完成后图片会自动保存到 `~/ComfyUI/output/`

### 步骤 8: 查看图片

打开 Finder 访问:
```
~/ComfyUI/output/
```

找到最新的 `news_*.png` 文件

## 🔧 故障排查

### Q: ComfyUI 无法访问？
**A**: 
```bash
# 检查 ComfyUI 是否运行
curl http://127.0.0.1:8188/system_stats

# 启动 ComfyUI
cd ~/ComfyUI
python main.py
```

### Q: 没有模型？
**A**: 
```bash
# 检查模型目录
ls ~/ComfyUI/models/checkpoints/

# 下载 SDXL 模型
# https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0
# 放到 ~/ComfyUI/models/checkpoints/
```

### Q: 生成速度慢？
**A**: 
- 降低 steps 到 15-20
- 使用更小的尺寸（如 512x256）
- 使用 Flux Turbo 模型（更快）
