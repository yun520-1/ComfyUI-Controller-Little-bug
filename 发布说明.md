## 🎉 新增功能

### 🦞 小龙虾智能批量生成

**核心功能：** 直接把文件扔给小龙虾，它自动帮你批量生成图片！

#### 支持的指令格式

1. **直接描述需求**
   ```
   帮我自动完成 10 个图片，图片结合最新新闻的观点，图片尺寸 1024*512
   ```

2. **上传小说/文本文件**
   ```
   [上传小说文件：《斗破苍穹.txt》]
   帮我把这本小说的每个章节生成一张插图，尺寸 1024*1024
   ```

3. **上传提示词列表**
   ```
   [上传文件：prompts.txt]
   用这个工作流帮我生成这些提示词的图片
   ```

#### 支持的文件类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `.txt` | 纯文本/提示词列表 | `prompts.txt` |
| `.md` | Markdown 文档 | `story.md` |
| `.docx` | Word 文档 | `小说.docx` |
| `.json` | 提示词 JSON | `prompts.json` |
| 链接 | 新闻链接 | `https://...` |

## 🚀 使用示例

### 示例 1：新闻配图

**指令：**
```
帮我生成 10 张图片，基于最新新闻，尺寸 1024*512
```

**执行流程：**
1. 获取今日热点新闻（10 条）
2. 为每条新闻生成画面描述
3. 使用 Z-Image 工作流生成
4. 保存到：`~/Downloads/comfyui_organized/news/日期/`

**预计耗时：** 约 30-40 分钟（10 张）

---

### 示例 2：小说插图

**指令：**
```
[上传《斗破苍穹.txt》]
帮我把这本小说的每个章节生成一张插图，玄幻修仙风格
```

**执行流程：**
1. 读取小说，识别章节
2. 提取每章关键场景
3. 生成玄幻风格提示词
4. 批量生成插图
5. 保存为：`斗破苍穹_第 01 章_萧炎突破.png`

**预计耗时：** 每章约 3-5 分钟

---

### 示例 3：提示词批量

**指令：**
```
[上传 prompts.txt]
用这个列表帮我生成图片，每个提示词生成 2 张不同种子的图
```

**prompts.txt 内容：**
```
一个亚洲女性，肖像，柔和光线
赛博朋克街道，霓虹灯，雨天
中国龙，云海，金色鳞片
...
```

## 🔧 技术细节

### 自动处理流程

```
用户指令
   ↓
[1] 解析意图 → 识别批量生成需求
   ↓
[2] 读取文件 → 提取内容/提示词/章节
   ↓
[3] 生成提示词 → AI 优化每个场景描述
   ↓
[4] 创建工作流 → 配置 ComfyUI API
   ↓
[5] 批量提交 → 逐个执行生成任务
   ↓
[6] 监控进度 → WebSocket 实时反馈
   ↓
[7] 下载整理 → 分类保存 + 元数据
   ↓
[8] 汇报结果 → 汇总成功/失败统计
```

### 依赖要求

- Python 3.8+
- ComfyUI (运行中)
- requests
- websocket-client
- Flask-SocketIO (Web 服务器)

## 🎯 最佳实践

### 推荐做法

1. **明确数量**：「帮我生成 10 张图片」
2. **指定尺寸**：「尺寸 1024*512」
3. **说明风格**：「玄幻修仙风格」
4. **分批执行**：大量任务分批次（每次 10-20 个）

### 避免做法

1. ❌ 一次生成上百张（容易超时）
2. ❌ 文件过大（建议分章节）
3. ❌ 提示词过于简单（小龙虾会帮你优化）
