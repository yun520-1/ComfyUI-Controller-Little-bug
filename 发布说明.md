## 🎯 安装说明

### 用户安装

```bash
# 从 ClawHub 安装
clawhub install comfyui-controller

# 从 GitHub 安装
git clone https://github.com/YOUR_USERNAME/comfyui-controller.git
cd comfyui-controller
pip install -r requirements.txt
```

### 验证安装

```bash
# 运行发现工具

clawhub info comfyui-controller
```
