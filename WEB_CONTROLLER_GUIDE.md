# ComfyUI 网页版智能控制器

## ✅ 功能特点

1. **实时任务监控** - 显示 ComfyUI 当前运行/排队任务
2. **自动排队等待** - 有任务运行时自动等待
3. **任务间隔控制** - 默认间隔 1 分钟，可自定义
4. **网页界面** - 浏览器打开即可使用
5. **进度可视化** - 实时显示生成进度

## 🚀 快速启动

### 方式 1：一键启动（推荐）

```bash
cd ~/.jvs/.openclaw/workspace/ComfyUI-Controller-Little-bug
python3 comfyui_web_controller.py
```

**会自动：**
1. ✅ 启动网页服务器（端口 8189）
2. ✅ 自动打开浏览器
3. ✅ 显示控制界面

### 方式 2：手动打开

```bash
# 1. 启动控制器

# 2. 打开浏览器访问

```

## 📋 使用界面

### 状态栏
- **ComfyUI 状态** - 在线/离线
- **运行中任务** - 当前正在运行的任务数
- **排队任务** - 等待中的任务数
- **已完成** - 已完成的任务数

### 任务配置
- **生成类型** - 选择图片类型（搞笑/人像/风景等）
- **生成数量** - 1-10 张
- **自定义主题** - 可选的自定义提示词
- **任务间隔** - 每个任务之间的等待时间（秒）

### 任务队列
- 显示所有任务
- 实时状态更新（等待中/运行中/完成/失败）
- 进度条显示整体进度

### 运行日志
- 实时显示运行日志
- 成功/错误/警告信息

## 💡 工作流程

```
1. 打开网页 → http://127.0.0.1:8189
2. 选择生成类型（如：搞笑幽默）
3. 设置生成数量（如：3 张）
4. 设置任务间隔（如：60 秒）
5. 点击"开始生成"
6. 自动执行：
   - 检查 ComfyUI 是否空闲
   - 提交第一个任务
   - 等待完成
   - 等待间隔时间
   - 提交下一个任务
   - ...
7. 所有任务完成后显示汇总
```

## 🔧 高级配置

### 修改默认端口

编辑 `comfyui_web_controller.py`，修改：
```python
CONTROLLER_PORT = 8189  # 改为其他端口
```

### 修改任务间隔

在界面上直接设置，或编辑代码：
```python
TASK_INTERVAL = 60  # 秒
```

### 修改分辨率

编辑工作流配置：
```python
"6": {"class_type": "EmptyLatentImage", "inputs": {
    "batch_size": 1, 
    "height": 512,  # 修改高度
    "width": 1024   # 修改宽度
}}
```

## ⚠️ 注意事项

1. **确保 ComfyUI 运行** - 控制器需要 ComfyUI 在 8188 端口
2. **任务间隔建议** - 至少 30 秒，避免 GPU 过载
3. **浏览器兼容性** - 推荐 Chrome/Safari/Edge
4. **不要关闭终端** - 关闭会停止网页服务器

## 🔧 故障排除

### 问题 1：无法打开浏览器

**解决：** 手动访问 http://127.0.0.1:8189

```bash
# 启动 ComfyUI
cd /Users/apple/Documents/lmd_data_root/apps/ComfyUI
python main.py --listen 0.0.0.0 --port 8188
```

**解决：** 修改 `CONTROLLER_PORT` 为其他端口
