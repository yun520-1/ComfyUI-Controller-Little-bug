# 🦞 小龙虾快速入门指南

> 3 分钟上手，开始批量生成图片！

---

## 🎯 第一步：确认环境

### 检查 ComfyUI 是否运行

```bash
curl http://127.0.0.1:8188/system_stats
```

如果返回 JSON 数据，说明 ComfyUI 正在运行 ✅

如果没有运行，启动它：

```bash
cd /Users/apple/Documents/lmd_data_root/apps/ComfyUI
./venv/bin/python run.py --listen 0.0.0.0 --port 8188
```

---

## 🚀 第二步：开始生成

### 方式一：直接指令（最简单！）

直接告诉小龙虾你要什么：

```
帮我生成 10 张图片，基于最新新闻，尺寸 1024*512
```

小龙虾会自动：
1. 获取最新热点新闻
2. 为每个新闻生成提示词
3. 批量执行生成
4. 保存并汇报结果

---

### 方式二：上传文件

#### 上传小说/文本

1. 发送文件：《斗破苍穹.txt》
2. 发送指令：「帮我把这本小说的每个章节生成一张插图，玄幻修仙风格」

#### 上传提示词列表

1. 创建 `prompts.txt`：
```
一个美丽的女孩，高清，精致
赛博朋克城市，霓虹灯
中国古风山水，水墨画风格
...
```

2. 发送文件 + 指令：「用这个列表帮我生成图片」

---

## 📁 第三步：查看结果

生成完成后，小龙虾会告诉你保存位置：

```
💾 结果已保存到：~/Downloads/comfyui_organized/news/2026-03-15/
```

查看图片：

```bash
ls -lh ~/Downloads/comfyui_organized/news/$(date +%Y-%m-%d)/
```

## 🎨 常用指令模板

### 新闻配图

```
帮我生成 [数量] 张图片，基于最新新闻，尺寸 [宽]*[高]
```

**示例：**
```
帮我生成 10 张图片，基于最新新闻，尺寸 1024*512
```

---

### 小说插图

```
[上传小说文件]
帮我把这本小说的每个章节生成一张插图，风格是 [风格]，尺寸 [宽]*[高]
```

**示例：**
```
[上传《斗破苍穹.txt》]
帮我把这本小说的每个章节生成一张插图，风格是玄幻修仙，尺寸 1024*1024
```

---

### 提示词批量

```
[上传 prompts.txt]
用这个列表帮我生成图片，每个提示词生成 [数量] 张不同种子的图
```

---

### 指定参数

```
帮我生成 [数量] 张图片，基于 [来源]
- 尺寸：[宽]*[高]
- 步数：[steps]
- 风格：[风格]
- 工作流：[工作流名称]
```

**示例：**
```
帮我生成 10 张图片，基于最新新闻
- 尺寸：1024*512
- 步数：20
- 风格：写实摄影
- 工作流：Z-Image Turbo
```

## ⚡ 高级技巧

### 技巧 1：分批生成

大量任务分批次执行，避免超时：

```
先帮我生成前 10 章的插图
...（完成后）
再帮我生成第 11-20 章的插图
```

---

### 技巧 2：指定工作流

如果有自定义工作流：

```
[上传小说.txt]
[上传 workflow.json]
用这个工作流帮我把小说的每个场景生成图片
```

---

### 技巧 3：中断任务

需要停止时，直接说：

```
停止生成
```

## 🔧 常见问题

### Q: 生成失败怎么办？

**检查清单：**
- [ ] ComfyUI 是否运行？
- [ ] 模型文件是否存在？
- [ ] 显存是否足够？

**解决方案：**
```bash
# 检查 ComfyUI 状态
curl http://127.0.0.1:8188/system_stats

# 检查模型目录
ls -lh /Users/apple/Documents/lmd_data_root/apps/ComfyUI/models/
```

---

### Q: 如何查看生成进度？

小龙虾会实时汇报：

```

   进度：16% (1/6)
```
