## 💡 使用示例

### 示例 1：基于新闻生成图片

```
1. 点击"🔄 刷新新闻"
2. 等待新闻加载（5 个类别）
3. 点击"📰 使用新闻生成提示词"
   或点击单个新闻使用
4. 选择尺寸：1920x1080
5. 设置数量：5
6. 设置间隔：30 秒
7. 点击"🚀 开始生成"
```

**生成的提示词示例：**
```
两会召开，人民大会堂，代表们讨论国家发展，庄严隆重，新闻插画风格，专业，高质量
```

```

2. 点击"搜索"
3. 在自定义提示词输入：
   cyberpunk Tokyo street, neon lights, rainy night
4. 附加要求：人群、车辆、蒸汽
5. 选择尺寸：2560x1080 (超宽)
6. 点击"🚀 开始生成"
```
